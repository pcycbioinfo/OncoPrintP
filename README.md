# OncoPrintP
OncoPrint++ is a web-based analytics pipeline by R Shiny. It functions as an all-in-one tool and integrates traditional OncoPrint/heatmap with enhanced functions and features for processing, visualizing and analyzing clinicogenomics data in oncology. It could be extended for general clinical bioinformatics and genomics.

# Installation
You can install OncoPrint++ from GitHub using the remotes package:

# Install devtools if you haven't already
install.packages("remotes")

# Install OncoPrint++
remotes::install_github("abbvie-external/OncoPrintP")
